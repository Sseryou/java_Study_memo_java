쿠키(Cookie)
- 개인화된 서비스(브라우저별 서비스)를 제공을 하기위한 기술
- 브라우저별 데이터 저장 기술
- 저장 데이터를 서버가 서비스를 제공하기 위해서 사용
	(개별화된 서비스)
- 쿠키 데이터는 브라우저에 저장, 실제 사용하는 쪽은 서버가 사용
- 요청시 서버로 전송(요청 헤더)
	: 요청 헤더 - cookies
	
민감한 정보를 담으면 안된다.
보안이 취약하므로, 보안이 매우 중요하다.
	ex)String value = AES256.encrypt(cookie.getValue());
	
	서버가 응답함으로써 브라우저가 저장한다

1. 쿠키의 동작과정
1) 생성
	서버의 응답 헤더 - Set-Cookie : 키=값; ....
		
2) 저장
	브라우저 -> 쿠키 저장(키=값)
	
3) 전송
	요청 -> 브라우저 -> 요청 헤더(cookies)를 통해 서버 전송
	
쿠키 생성 메서드
HttpServletResponse
	addCookie(Cookie cookie) : 응답 헤더 Set-Cookie ...


쿠키를 조회하는 메서드

	Cookie[] getCookies();
